# MoodTrack

MoodTrack is a Flutter application designed to help users track their emotional well-being. It incorporates psychological principles and AI-driven insights to provide personalized reflections and support.

## Features

- **Mood Tracking**: Log your daily mood with an intuitive emoji interface.
- **Contextual Insights**: Answer simple questions to provide context to your feelings.
- **AI Reflections**: Receive personalized, empathetic feedback based on your entries.
- **History & Trends**: Visualize your mood evolution over time with interactive charts.
- **Smart Analysis**: Get weekly insights about your emotional patterns.

## Design System

The app follows a calm and welcoming design language:
- **Palette**: Mint Green, Lavender, Soft Blue, Cream White.
- **Typography**: Inter font for readability and modern feel.
- **Components**: Custom `MoodCard`, `MoodButton`, and `MoodIcon` widgets.

## Tech Stack

- **Flutter**: UI Framework.
- **Google Fonts**: Typography.
- **FL Chart**: Data visualization.
- **Provider**: State management.
- **Intl**: Date formatting.

## Getting Started

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to start the app.
